Calcular Cotización

Crear una cotización de precios de los componentes de una computadora, a partir de tres objetos de tipo SELECT que nos permiten seleccionar:

  1. Procesador (Intel I3 - $450, Intel I5 $550, Intel I7 $800).
  2. Monitor (Samsung 20' - $250, Samsung 22' - $350, Samsung 26' - $550).
  3. Disco Duro(500 Gb - $300, 1 Tb - $440, 3 Tb - $500).

Para cada característica indicamos string a mostrar (Ej. Intel I3) y el valor asociado ha dicho string (Ej. 450).

Al presionar un botón "Calcular" mostrar el presupuesto en un objeto de tipo TEXT.
